Self-Hosting & Setup Guide

This guide outlines the step-by-step configuration required to host, execute, and maintain the Count Check platform inside a Windows headless server or local desktop environment.

Note

Make sure your Python installation environment is added to your system's PATH variables to allow execution from absolute CLI scripts.

Prerequisites & Dependencies

To host the platform, you must have Python 3.12 or higher installed. The system relies on several standard and third-party libraries. Open your command prompt and run the following command to initialize your virtual environment and install the required modules:

pip install discord.py pillow matplotlib numpy scipy httpx

Directory Structure & Assets

The directory tree must be structured exactly as follows to ensure relative file paths resolve correctly at runtime:

├── main.py                     # Primary Bootloader

├── core/

├── storage/
│    └── database.py            # SQLite thread-isolated pool & WAL migration
├── models/

│    └── trade_plan.py          # Position output containers
├── engine/
│    ├── analytics_engine.py    # Risk models and decay calculations

Configuration & Initialization Sequence

  1. Set Environment VariablesCreate a system environment variable or configure your Windows startup script to export your private Discord Bot Token:set DISCORD_BOT_TOKEN=YOUR_PRIVATE_TOKEN_HERE
  2. First Boot SequenceExecute python main.py in your terminal. On boot, the system automatically checks for database files and directory structures. If market_history.db or osrs_core/data/items.json are missing or corrupted, the system executes self-healing migrations, extracts standard templates, and forces a bulk download of OSRS item mappings from the RuneScape Wiki CDN.
  3. Registering CommandsOnce the bot is active inside your Discord server, execute the prefix command !sync (or !globalsync) as the Bot Owner. This purges the old application command cache globally and locally, registering your slash command tree instantly. Reload your Discord client (Ctrl + R) to update the autocomplete menu.
