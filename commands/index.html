Administrative & Prefix Commands (Owner Only)

These prefix commands bypass the public slash tree entirely. They can only be executed by the bot owner inside your private administrator channel.

!globalsync

Forces a complete synchronization of the bot's application command trees across Discord's gateways. Run this after updating command parameters.

!botreset

Safely purges all simulated transaction histories, open orders, and trade balances from the SQLite database. Resets mock capital back to 10,000,000 GP.

!debugview

Queries and lists all active, unresolved tester bug reports and suggestions compiled from the database.

!debugresolve <id>

Deletes a completed or addressed report from the database by its unique entry ID.

!post <#channel> <include_button_bool> <Title> | <Description>

Utility to post a custom embed to any server channel with the option to attach the persistent green Verify Agreement button. Use | to separate the title and description blocks.
